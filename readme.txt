Extend Smartling Connector functionality by adding localization for AdRotate plugin

== Description ==

[The Smartling Connector](https://wordpress.org/plugins/smartling-connector) extends the WordPress interface for seamless management of the translation process. This plugin allows to localize [AdRotate](https://wordpress.org/plugins/adrotate/) adverts and groups

Integration Features

* Automatic change detection for content updates
* Robust custom workflow engine configurable per language
* Automatic download of completed translations to WordPress
* Translation Memory integration
* No tie-ins to translation agencies or vendors
* Reporting for translation velocity, efficiency

== Installation ==

= Minimum Requirements =
* WordPress 4.6 or higher
* [Smartling Connector](https://wordpress.org/plugins/smartling-connector) 4.1 or higher

1. Upload the plugin files to the `/wp-content/plugins/smartling-adrotate-localization` directory, or install the plugin through the WordPress plugins screen directly.
1. Go to the Plugins screen and **Network Activate** the AdRotate localization plugin.
